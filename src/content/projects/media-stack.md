The classic homelab rite of passage, done properly: a containerised media
server stack that handles streaming to every screen in the house, a request
system so the household can ask for things without asking me, and the
automation glue that keeps it all fed and organised.

It's also the project that taught me the operational side of self-hosting —
reverse tunnels, container networking, storage planning, and the hard lesson
that the family treats it as production infrastructure whether I do or not.

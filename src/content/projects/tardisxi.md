A fan-made FFXI server that I host and maintain myself — not because the world
needed another one, but because I wanted to understand the whole stack from the
inside. Running it means wrangling Lua scripting, a genuinely large MySQL
database, decades-old game logic, and a production Linux box that real people
log into and expect to work.

It has taught me more about databases, backups, uptime, and debugging other
people's code than any course would have. It breaks. I fix it. It breaks
differently. That's the deal.
